# 🎯 InternAI
> Automated Internship Scraper & Smart Tracker

## Features
- Auto-scrapes Internshala every 6 hours
- AI-style scoring system (0-100)
- Telegram notifications for top internships
- Beautiful dark theme dashboard
- Filter, search and sort internships

## Setup
1. npm install
2. cp .env.example .env (add your Telegram credentials)
3. node server.js
4. Open http://localhost:3000

## API
- GET /api/internships
- POST /api/scrape
- GET /api/stats
- DELETE /api/internships/:id
